Products

As the world becomes more and more automated thanks to artificial intelligence and machine learning, data collection and analysis is more complex than ever before – at SequenceIQ we have a clear roadmap to offer an answer for all who seek a solution for their increasing big data problems.

For organizations that want to combine spatial, sensor and other historical or streaming based large data sets and discover relationships on massive amounts of multidimensional data, we offer an easy and intuitive solution with our services and product stack.

xTractCore

Core is our open source, cost effective way to start and run multiple instances and versions of Hadoop clusters in the cloud or on-premise. It is a cloud agnostic and cost effictive Hadoop As-a-Service platform. Provides automatic scaling, secure multi tenancy and full cloud lifecycle management.

The building blocks of the xTract Core platform are spread across three different layers.

Infrastructure We support a cloud based deployment model and an on-premise deployment running on physical hardware, and different virtualizations.

Data Fabric We break the data silos created by different departments and build 'data lakes' by offering a centralized view on different underlying datasources.

Data insights We allow enterprises to run machine learning, map-reduce jobs and data processing workflows to access data insights.

Note: xTract Core is cloud and Hadoop provider agnostic. We encourage our customers to deploy the platform on their preferred cloud provider*. We believe that vendor locking seize innovation, thus we support and are partners with different Hadoop vendors - Apache Hadoop, Hortonworks Data Platform, Cloudera Distribution Hadoop.

* the autoscaling feature might not work with all cloud providers

xTractSpacetime

The idea behind Spacetime is that big data and actionable insights are not limited to expert organizations and departments but can be provided to all parts of a business in a self-service manner. Big Data development has never been easier.

Spacetime helps enterprise application, web, mobile, 'Internet of Things', and iBeacon developers to build data driven applications, interact and learn from their data with ease. We offer an intuitive UI and a simple and a secure REST API to configure data processing workflows, ETL processes, gain deeper insight by running machine learning algorithms and aggregate the results into a common 360 view of the enterprise. Once these semantic relationships are discovered the platform allows to configure rules, actions and notifications to react on external events and patterns identified on streaming data. We support a growing set of commercial data connectors, and provide a connector SDK to build your own.

The building blocks of the xTract Spacetime platform are the following.

Data Workflow We allow developers or business analysts to build their data and job workflow based on their organization's needs and individual use cases using our pre-developed building block (ML algortihms, MapReduce jobs, data patterns. The workflow is a collection of procedural steps, interactions, input and output - steps needed to describe a big data business process. Supports multiple running workflow instances, A/B testing, training and evaluation. Workflow instances can be scheduled, triggered by external events or manually.

Actionable Insights We allow developers to set up rules, associate actions to rules and raise notifications on different output channels. As the data is streaming through the platform it interacts with the preconfigured rules and action pairs and reacts at millisecond latency - learning insights and acting on big data triggered rules has never been easier, all these at large scale.

Data Connectors We support a large set of data connectors from open source standards to proprietary ones, and provide an SDK to build custom connectors.
